## Overview
A Python console application that recommends songs based on the user's current mood. The system includes a curated database of 70+ songs from various genres and time periods, including trending 2023-2024 hits.

## Features
- 6 mood categories (Happy, Sad, Energetic, Calm, Romantic, Angry)
- Genre filtering (Pop, Rock, Hip-Hop, Electronic, Indie, Jazz, Classical)
- Year-based filtering (2024, 2023, 2020s, 2010s, Classic Hits)
- Customizable number of recommendations (1-50 songs)
- Trending song indicators
- Popularity ratings for each song

## Technologies Used
- **Language:** Python 3.x
- **Data Structures:** Dictionaries, Lists
- **Libraries:** None (pure Python)

## How to Use
1. Launch the program
2. Select your current mood 
3. Choose genre filter
4. Choose year filter 
5. Enter number of songs you want
6. View your personalized playlist!
