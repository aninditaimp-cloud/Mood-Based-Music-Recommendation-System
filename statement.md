# Problem Statement
Many people struggle to find music that matches their current emotional state. Existing music platforms have complex interfaces and don't provide quick, mood-based recommendations.

## Scope
This project provides a simple, console-based music recommendation system that:
- Allows users to select their current mood
- Filters songs by genre and release year
- Displays curated song recommendations instantly

## Target Users
- Music enthusiasts looking for mood-appropriate songs
- Students seeking study/relaxation music
- Anyone wanting quick music recommendations without complex apps

## High-Level Features
1. Mood-based song categorization
2. Multi-filter search system
3. Trending song highlights
4. Popularity-based ranking
5. User-friendly console interface
